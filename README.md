After installing neovim

- Create config directory `mkdir -p ~/.config/nvim`

- Create init.lua file `touch init.lua`
- Create lua directory `mkdir -p nvim/lua/chris`
- Inside `chris`
    - `touch lazy.lua`
    - `mkdir core`
    - `mkdir plugins`
